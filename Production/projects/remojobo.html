RemoJobo is a job board with a twist, we feature only jobs that you can do holistically from home, or the majority. The application is build on wordpress and uses ifttt to scrape various RSS feeds for specified keywords, then populates the wordpress blog with only remote Jobs. My Goal was to build something with “legs”.
